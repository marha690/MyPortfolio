Global illumination with ray tracing

Implemented in C++ with GLM

About this project

This is a implementation of a basic Whitted ray-tracer.

Ray tracing is a method used to make a global illumination model for a scene. The rays transport light around the scene by bouncing when hitting surfaces. How the ray behaves are depending on the surfaces which it hits, and the lights properties inside the ray changes when it hits a surface. The rays start inside the camera and goes through pixels inside the view plane. Those rays in the scene which is spawned from a ray going through a pixel in the view plane will only contribute to computations for that pixel. Since the computers power is limited the number of rays cannot be infinitive, and the Russian roulette method is used to limit them.
