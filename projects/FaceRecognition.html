Face recognition with eigenfaces

In images

Face detection

To test the program images from the Caltech database of faces from 1999 is used. The first step is to detect and isolate faces inside the images.

Our approach was to first make a face mask to find the face. After that a eye map is combined for a safer detection and also as anchors for the image adjustments. The images adjustments are scaling, rotatating and cropping so that only the face exists. Color adjustments are also applied.

Face recognition

With the images prepared an database with eigenfaces are made. In the database there are 16 different persons which has unique id's. Every image in the database has the same id as the person inside that image.

To be able to match faces with an test image an principal component analyzis are used, where the dimensions in the eigenfaces are made to represent faces best.
